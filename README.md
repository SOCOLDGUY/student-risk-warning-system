# 学生学业风险预警系统

## 📋 项目简介
基于Jupyter Notebook实现的学生学业风险预警系统，使用多种机器学习模型预测学生挂科风险，并提供个性化改进建议。

## 🎯 项目功能
1. 数据生成：模拟3000名学生的学习行为数据
2. 模型训练：对比逻辑回归、随机森林、XGBoost三种模型
3. 风险评估：预测学生的低风险、中风险、高风险等级
4. 报告生成：为每位学生生成个性化改进建议报告

## 📁 文件说明
- `student_risk_analysis.ipynb` - 完整的主代码文件（Jupyter Notebook格式）
- `student_performance_strong.csv` - 生成的3000名学生数据集
- `model_performance_v2.png` - 模型性能对比图
- `feature_importance.png` - 特征重要性分析图
- `student_report_*.txt` - 个性化学生报告
- `system_summary.txt` - 系统总结报告
